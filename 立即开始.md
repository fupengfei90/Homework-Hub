# 🚀 立即开始预览

## 三步快速预览

### 第1步：打开微信开发者工具
1. 如果没有安装，先下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具

### 第2步：导入项目
1. 点击左上角"+" → 选择"导入项目"
2. 项目目录选择：`/Users/fupengfei/Documents/ws/Homework-hub/Homework-Hub`
3. **AppID选择"测试号"**（不需要真实AppID，可以直接预览）
4. 点击"导入"

### 第3步：编译运行
1. 点击工具栏的"编译"按钮（或按快捷键 Ctrl+B / Cmd+B）
2. 在左侧模拟器中查看效果！
