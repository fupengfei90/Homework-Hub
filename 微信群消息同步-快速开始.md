# 微信群消息同步功能 - 快速开始指南

## 🎯 功能说明

通过itchat监听微信群消息，自动同步到小程序展示，支持作业和通知分类。

## 📋 前置条件

- ✅ Python 3.6+ 已安装
- ✅ 微信开发者工具已安装
- ✅ 云开发已开通

---

## 🚀 5分钟快速部署

### 第一步：部署云函数（5分钟）

1. 在微信开发者工具中打开项目
2. 找到 `cloudfunctions/syncMessage` 文件夹
3. 右键 → "上传并部署：云端安装依赖"
4. 等待部署完成

**⚠️ 重要：配置HTTP触发器**

由于你没有云服务器，必须配置HTTP触发器：

1. 打开[云开发控制台](https://console.cloud.tencent.com/tcb)
2. 进入你的环境 → 云函数 → syncMessage
3. 点击"触发器" → "添加触发器"
4. 选择"HTTP触发器"，设置路径为 `/syncMessage`
5. 复制触发器的访问URL（类似：`https://xxx-xxx.service.tcloudbase.com/syncMessage`）

## 📱 使用说明

### 查看同步消息

1. 打开小程序
2. 点击底部导航"💬 班级消息"
3. 可以筛选查看"全部"、"作业"、"通知"

### 筛选消息类型

- **全部**：显示所有消息
- **作业**：只显示作业相关消息
- **通知**：只显示通知相关消息

### 复制消息

点击消息的"复制"按钮，可以复制消息内容。

### 删除消息

点击消息的"删除"按钮，可以删除不需要的消息。

## 🐛 常见问题

### Q1: 扫码登录后提示"账号异常"

**A：**
- 建议使用新注册的微信小号
- 不要使用主号，避免被封
- 确保网络正常

### Q2: 消息无法同步到小程序

**A：检查以下内容：**

1. **云函数是否部署成功？**
   - 打开云开发控制台 → 云函数 → syncMessage
   - 查看是否有部署记录

2. **HTTP触发器URL是否正确？**
   - 检查 `wechat_sync.py` 中的 `HTTP_API_URL`
   - 确保URL可以访问（可以用浏览器打开测试）

3. **网络是否正常？**
   - 确保脚本运行时网络畅通
   - 查看脚本终端是否有错误信息

4. **数据库是否创建？**
   - 打开云开发控制台 → 数据库
   - 确认 `sync_messages` 集合已创建

### Q3: 如何让脚本一直运行？

**A：推荐方案：**

#### 方案1：使用云服务器（推荐）

```bash
# 1. 上传脚本到服务器
scp -r itchat-sync/ user@your-server:/home/

# 2. 在服务器上安装依赖
ssh user@your-server
cd /home/itchat-sync
pip3 install -r requirements.txt

# 3. 使用nohup后台运行
nohup python3 wechat_sync.py > sync.log 2>&1 &

# 4. 查看运行状态
tail -f sync.log
```

#### 方案2：使用Screen（Mac/Linux）

```bash
# 安装screen
brew install screen  # Mac
sudo apt-get install screen  # Ubuntu

# 创建新窗口
screen -S wechat-sync

# 运行脚本
python wechat_sync.py

# 退出窗口（脚本继续运行）
按 Ctrl+A 然后按 D

# 重新进入窗口
screen -r wechat-sync
```
