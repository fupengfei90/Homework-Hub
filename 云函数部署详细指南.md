# 云函数部署详细步骤指南

## ❌ 错误信息

```
Error: 请在编辑器云函数根目录（cloudfunctionRoot）选择一个云环境
```

**原因：** 上传云函数前，需要先在云开发控制台选择云环境。

---

## 🔧 正确的部署步骤

### 第1步：打开云开发控制台并选择环境

1. **打开云开发控制台**
   - 在微信开发者工具中点击顶部的 **"云开发"** 按钮
   - 会打开云开发控制台网页

2. **选择云环境**
   - 在云开发控制台顶部，点击环境选择器
   - 选择你的环境：`cloud1-0g5wc99td84df799`
   - **关键：必须先选择环境！**

3. **验证环境已选择**
   - 环境名称应该显示在控制台顶部
   - 环境ID：`cloud1-0g5wc99td84df799`

---

### 第2步：部署 syncMessage 云函数

1. **回到微信开发者工具**

2. **找到云函数文件夹**
   - 在左侧文件树中找到 `cloudfunctions/` 文件夹
   - 展开该文件夹
   - 找到 `syncMessage/` 子文件夹

3. **右键点击 syncMessage 文件夹**

4. **选择部署选项**
   - 选择 **"上传并部署：云端安装依赖"**
   - **或者**选择 **"上传并部署：所有文件"**

5. **等待部署完成**
   - 控制台会显示部署进度：
     ```
     正在上传文件...
     正在部署...
     部署成功
     ```
   - 部署时间：约1-2分钟

---

### 第3步：部署 aiParse 云函数

1. **在文件树中找到 `cloudfunctions/aiParse/` 文件夹**

2. **右键点击 aiParse 文件夹**

3. **选择部署选项**
   - 选择 **"上传并部署：云端安装依赖"**

4. **等待部署完成**
   - 等待约1-2分钟

---

### 第4步：验证云函数部署成功

1. **打开云开发控制台**
   - 点击顶部的"云开发"按钮

2. **点击"云函数"**

3. **查看云函数列表**
   - 应该能看到以下2个云函数：
     - ✅ `aiParse`
     - ✅ `syncMessage`

4. **测试云函数**
   - 点击云函数名称
   - 点击"测试"
   - 输入测试数据
   - 点击"执行测试"

---

## 🔍 故障排查

### 问题1：部署选项是灰色的

**可能原因：**
1. 没有选择云环境
2. 云环境未开通
3. 云函数文件夹结构不正确

**解决方法：**

**步骤1：检查云环境是否开通**
1. 点击"云开发"按钮
2. 如果提示开通，点击"开通"
3. 等待开通完成

**步骤2：选择云环境**
1. 在云开发控制台顶部
2. 点击环境选择器
3. 选择 `cloud1-0g5wc99td84df799`

**步骤3：返回开发者工具重新部署**
1. 回到微信开发者工具
2. 右键云函数文件夹
3. 再次尝试部署

---

### 问题2：部署时提示"云函数根目录配置错误"

**检查配置：**
- `project.config.json` 第62行：`"cloudfunctionRoot": "cloudfunctions/"`
- 确保路径正确

**验证：**
1. 在项目根目录下应该有 `cloudfunctions/` 文件夹
2. 该文件夹下应该有 `aiParse/` 和 `syncMessage/` 两个子文件夹

---

### 问题3：部署成功但调用时报错

**检查云函数代码：**
1. 打开云开发控制台
2. 点击"云函数" → `syncMessage`
3. 点击"代码"标签
4. 检查代码是否正确上传

**检查日志：**
1. 点击"日志"标签
2. 查看是否有错误信息

## 🎯 最简单的部署方法

### 方法1：通过云开发控制台上传（推荐）

1. **打开云开发控制台**
   - 点击微信开发者工具顶部的"云开发"按钮

2. **进入云函数页面**
   - 点击左侧菜单的"云函数"

3. **点击"新建"按钮**

4. **选择"上传本地云函数"**
   - 文件夹选择：`cloudfunctions/syncMessage`
   - 云函数名称：`syncMessage`（自动填充）
   - 运行环境：Node.js 16
   - 点击"上传"

5. **重复步骤4**
   - 上传 `cloudfunctions/aiParse`
   - 云函数名称：`aiParse`

### 方法2：通过微信开发者工具右键部署

1. **确保已选择云环境**
   - 打开云开发控制台
   - 选择环境：`cloud1-0g5wc99td84df799`

2. **返回微信开发者工具**

3. **右键云函数文件夹**
   - 右键 `cloudfunctions/syncMessage`
   - 选择"上传并部署：云端安装依赖"

4. **等待部署完成**

## ⚠️ 重要提示

### 提示1：必须先选择云环境

**错误做法：**
- 直接右键云函数文件夹部署
- 没有在云开发控制台选择环境

**正确做法：**
1. 先打开云开发控制台
2. 选择云环境
3. 再返回开发者工具部署

### 提示2：环境ID必须一致

确保：
- `app.js` 中的环境ID：`cloud1-0g5wc99td84df799`
- 云开发控制台选择的环境：`cloud1-0g5wc99td84df799`
- 两者必须一致！

### 提示3：云函数依赖安装

首次部署建议选择：
- **"上传并部署：云端安装依赖"**
- 会自动安装 `package.json` 中的依赖
- 避免手动安装依赖的麻烦

## 📞 需要帮助？

### 如果部署失败
1. 检查是否选择了云环境
2. 检查网络连接
3. 查看云开发控制台的错误提示
4. 尝试删除云函数后重新部署

### 如果调用失败
1. 查看云函数日志
2. 检查云函数代码
3. 验证环境ID是否正确
4. 重新部署云函数
