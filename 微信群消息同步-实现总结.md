## 🚀 部署步骤

### 步骤1：部署云函数
```bash
# 在微信开发者工具中
1. 右键 cloudfunctions/syncMessage
2. 选择 "上传并部署：云端安装依赖"
3. 等待部署完成
```

### 步骤2：配置HTTP触发器
```bash
# 在云开发控制台中
1. 打开 云函数 → syncMessage
2. 点击 "触发器" → "添加触发器"
3. 选择 "HTTP触发器"，路径设置为 /syncMessage
4. 复制触发器访问URL
```

### 步骤3：安装Python依赖
```bash
cd /Users/zhangqiaoling/ws/Homework-Hub/Homework-Hub/itchat-sync
```

## 🔧 核心功能说明

### 1. 消息监听
- 使用itchat监听微信群消息
- 支持模糊匹配群名和用户名
- 消息去重，避免重复同步

### 2. 智能识别
- 根据关键词自动识别消息类型
- 作业关键词：作业、练习、习题、预习、复习等
- 通知关键词：默认为通知

### 3. 数据同步
- 通过HTTP API同步到云函数
- 云函数存储到数据库
- 小程序从数据库读取展示

## 🐛 常见问题

### Q1: 扫码登录异常
**A：** 使用新注册的微信小号，不要使用主号

### Q2: 消息无法同步
**A：** 检查：
- 云函数是否部署成功
- HTTP触发器URL是否正确
- 网络是否正常
- 运行测试脚本排查

### Q3: 如何长期运行
**A：** 部署到云服务器或使用screen/tmux
