# 快速预览指南

## 方法一：使用测试号快速预览（推荐）

### 1. 打开项目
1. 打开 **微信开发者工具**
2. 点击"+" → "导入项目"
3. 选择项目目录：`/Users/fupengfei/Documents/ws/Homework-hub/Homework-Hub`
4. 在"AppID"处选择"测试号"（不需要真实AppID）
5. 点击"导入"

### 2. 配置项目（可选，用于完整功能）
- **如果只想看界面效果**：可以直接编译运行，界面会正常显示
- **如果要测试完整功能**：需要配置云开发（见下方说明）

### 3. 编译运行
- 点击工具栏的"编译"按钮
- 在模拟器中查看效果

## 方法二：使用真实AppID（完整功能）

### 1. 获取AppID
- 访问 [微信公众平台](https://mp.weixin.qq.com/)
- 注册小程序账号
- 获取 AppID

### 2. 配置项目
修改 `project.config.json`：
```json
{
  "appid": "你的AppID"
}
```

### 3. 开通云开发（必须，否则数据功能无法使用）
1. 在微信开发者工具中点击"云开发"按钮
2. 开通云开发服务
3. 创建云环境
4. 在云开发控制台 → 设置 → 环境设置中获取环境ID

### 4. 配置云环境（可选）
- 如果使用默认环境，可以不配置
- 如果需要指定环境，修改 `app.js` 中的 `env` 参数

### 5. 创建数据库
1. 云开发控制台 → 数据库 → 创建集合
2. 集合名称：`homework`
3. 权限：所有用户可读，仅创建者可写

## 常见问题

### Q: 编译后页面空白？
**A:** 检查控制台是否有错误，可能是图片路径问题。可以先添加占位图片。

### Q: 提示"云开发未初始化"？
**A:** 这是正常的，如果只想看界面，可以忽略。要使用数据功能需要开通云开发。

### Q: 标签栏图标不显示？
**A:** 需要在 `images/` 目录下添加图标文件，或暂时注释掉 `app.json` 中的 `tabBar` 配置。

### Q: 如何快速添加占位图标？
**A:** 可以创建简单的纯色PNG图片，或使用在线图标生成工具。
