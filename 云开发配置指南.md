# 云开发配置指南

## ❌ 错误信息

```
errCode: -601034 | errMsg: 没有权限，请先开通云开发或者云托管
```

这个错误说明**云开发未开通或未配置环境ID**。

---

## 🔧 完整配置步骤

### 第1步：开通云开发（必须）

在微信开发者工具中：

1. **点击顶部工具栏的"云开发"按钮**
   - 如果按钮是灰色的，说明你的小程序没有权限使用云开发
   - 需要在微信公众平台配置权限

2. **如果可以点击，会弹出云开发控制台**
   - 如果未开通，会提示"开通云开发"
   - 点击"开通"

3. **创建云环境**
   - 环境名称：`homework-hub`（可以自定义）
   - 环境ID：自动生成（例如：`homework-hub-7g8h9j0k`）
   - 基础版：免费额度够用
   - 按量付费：按实际使用量计费

4. **等待开通完成**
   - 通常需要1-3分钟

---

### 第2步：获取环境ID

开通后：

1. 在云开发控制台顶部可以看到你的环境ID
2. 格式通常是：`homework-hub-xxxxxxxx-xxxxx`
3. **复制这个环境ID**

---

### 第3步：配置环境ID

打开 `app.js` 文件，找到第16行：

```javascript
// app.js 第11-18行
wx.cloud.init({
  env: 'your-env-id', // 👈 这里需要替换为你的环境ID
  traceUser: true,
});
```

**替换为：**

```javascript
wx.cloud.init({
  env: 'homework-hub-7g8h9j0k', // 👈 替换为你的真实环境ID
  traceUser: true,
});
```

**或者留空使用默认环境：**

```javascript
wx.cloud.init({
  traceUser: true, // 不填env参数，使用默认环境
});
```

---

### 第4步：创建数据库集合

在云开发控制台 → 数据库中创建以下集合：

1. **homework** - 作业数据
2. **homeworkReminders** - 作业提醒
3. **sync_messages** - 同步消息（用于微信群消息同步）
4. **sync_config** - 同步配置（用于微信群消息同步）

---

### 第5步：部署云函数

在微信开发者工具中：

1. 找到 `cloudfunctions` 文件夹
2. 右键每个云函数文件夹
3. 选择"上传并部署：云端安装依赖"
4. 等待部署完成

需要部署的云函数：
- ✅ `aiParse` - AI识别图片
- ✅ `syncMessage` - 消息同步（新增）

---

## 🚀 快速配置方案（推荐）

如果你还没有开通云开发，或者不确定如何配置，可以使用以下**快速方案**：

### 方案：不使用云函数，使用模拟数据

小程序代码已经支持**模拟数据**模式，即使不开通云开发也可以正常使用基础功能。

**修改 `app.js`：**

```javascript
wx.cloud.init({
  // env: 'your-env-id', // 注释掉这行
  traceUser: true,
});
```

这样小程序会：
- ✅ 使用本地模拟数据（作业列表、发布作业等功能可用）
- ❌ 云函数功能不可用（AI识别、消息同步等）

**查看模拟数据位置：**

1. `pages/index/index.js` - 作业列表有模拟数据
2. `pages/detail/detail.js` - 详情页有模拟数据

## 💡 常见问题

### Q1: "云开发"按钮是灰色的，点击不了

**A：**
1. 检查小程序AppID是否正确
2. 登录[微信公众平台](https://mp.weixin.qq.com)
3. 进入"开发" → "开发管理" → "开发设置"
4. 确认"云开发"功能已开启

### Q2: 开通云开发需要付费吗？

**A：**
- 基础版：免费额度（每月2GB数据库，5万次读写）
- 按量付费：超出免费额度后按量计费
- 个人学习使用，免费额度完全够用

### Q3: 不想开通云开发，可以吗？

**A：**
- 可以！使用模拟数据模式
- 基础功能（作业列表、发布作业）可用
- 高级功能（AI识别、消息同步）不可用
- 修改 `app.js` 注释掉 `env` 参数即可
