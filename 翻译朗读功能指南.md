# 文本→翻译→朗读功能配置指南

本文档介绍"作业助手"小程序的"文本→翻译→朗读"完整功能链路。

## ⚠️ 重要提示

**当前版本说明**:
- 微信小程序对HTTP/HTTPS音频播放有严格限制
- 外部TTS API(有道、百度)返回的音频无法在微信小程序中直接播放
- 当前使用浏览器SpeechSynthesis作为主要方案
- 该方案仅在某些环境(如开发者工具)中可用

**在真机上的表现**:
- 语音朗读功能可能不可用
- 建议作为辅助功能,不要作为核心功能
- 后续可考虑接入微信同声传译插件

## 一、功能概述

本功能实现了完整的"文本→翻译→朗读"工作流程:

```
文本输入 → 翻译API → 语音合成(TTS) → 音频播放
```

### 核心特性

1. **双模式朗读**
   - 📝 **原文模式**: 直接朗读中文作业内容
   - 🌐 **翻译模式**: 先翻译成目标语言,再朗读译文

2. **多语言支持**
   - 🇨🇳 中文→英文翻译
   - 🇬🇧 中文→中文(方言朗读,可选)
   - 未来可扩展更多语言

3. **智能TTS**
   - 自动识别文本语言
   - 支持中英文混合朗读
   - 使用免费在线TTS服务

### 2. 核心API

#### 翻译API
- **当前使用**: 有道翻译API (免费)
- **地址**: `https://fanyi.youdao.com/openapi.do`
- **特点**: 无需注册,支持中英互译

#### TTS API
- **当前使用**: 有道TTS API (免费)
- **地址**: `http://tts.youdao.com/fanyivoice`
- **特点**: 支持中英文,无需注册

#### 未来可接入
- 腾讯云机器翻译API
- 腾讯云语音合成API
- 百度翻译API
- 百度TTS API

## 三、功能使用

### 1. 原文朗读模式

**适用场景**: 朗读中文作业,帮助孩子理解作业内容

**使用步骤**:
1. 打开作业详情页
2. 确保朗读模式为"原文"
3. 点击"点我读作业"大按钮
4. 系统自动朗读作业标题和内容

**特点**:
- ✅ 直接朗读中文
- ✅ 无需翻译,速度快
- ✅ 适合中文听力练习

### 2. 翻译朗读模式

**适用场景**: 学习英语,提升外语能力

**使用步骤**:
1. 打开作业详情页
2. 点击朗读模式,切换为"翻译"
3. 选择目标语言(英语/中文)
4. 点击"读翻译内容"按钮
5. 系统先翻译,再朗读译文

**特点**:
- ✅ 先翻译再朗读
- ✅ 支持中英互译
- ✅ 提升外语学习效果

### 3. 语言切换

**支持的语言对**:
- 🇨🇳 中文 → 🇬🇧 英语
- 🇨🇳 中文 → 🇨🇳 中文(未来支持方言)

**切换方法**:
1. 进入翻译模式
2. 在"目标语言"区域点击选择
3. 系统保存选择,下次使用

## 四、技术实现

### 1. 核心代码结构

#### 翻译服务 (`tencent-tts-service.js`)

```javascript
// 文本翻译
function textTranslate(text, source, target)

// 文本转语音
function textToVoice(text, voiceType, speed, volume)

// 完整流程: 翻译+朗读
async function translateAndSpeak(text, sourceLang, targetLang, needTranslate)
```

#### 详情页逻辑 (`detail.js`)

```javascript
// 切换朗读模式
switchReadMode(e)

// 切换目标语言
switchTargetLang(e)

// 开始朗读(支持原文/翻译)
async startReading()
```

### 2. 数据流程

```
用户点击朗读
    ↓
检查朗读模式(原文/翻译)
    ↓
如果是翻译模式:
    ↓
    调用翻译API → 获取译文
    ↓
调用TTS API → 获取音频URL
    ↓
创建音频播放器 → 播放音频
```

## 五、配置说明

### 1. 当前配置(免费版本)

当前使用的是免费API,无需配置:

```javascript
// 翻译API: 有道翻译(免费)
// TTS API: 有道TTS(免费)

// 无需配置密钥,开箱即用
```

### 2. 升级到腾讯云API(可选)

如果需要更高质量的翻译和TTS,可以升级到腾讯云API:

#### 步骤1: 获取密钥

1. 登录腾讯云控制台: https://cloud.tencent.com/
2. 开通"机器翻译"服务
3. 在"访问管理"中创建API密钥
4. 获取 SecretId 和 SecretKey

#### 步骤2: 配置密钥

打开 `utils/tencent-cloud-config.js`,填入密钥:

```javascript
const TENCENT_CLOUD_CONFIG = {
  secretId: '你的SecretId',
  secretKey: '你的SecretKey',
  region: 'ap-beijing',
  // ... 其他配置
};
```

#### 步骤3: 配置小程序域名

在小程序后台添加服务器域名:

```
request合法域名:
https://tmt.tencentcloudapi.com
```

## 六、故障排查

### 问题1: 翻译失败

**症状**: 点击朗读后提示"翻译服务暂不可用"

**原因**:
- 网络连接问题
- 有道API限流
- 请求参数错误

**解决方案**:
1. 检查网络连接
2. 稍后重试
3. 如果频繁失败,建议升级到腾讯云API

### 问题2: 音频无法播放

**症状**: 提示"音频播放失败"

**原因**:
- TTS API返回错误
- 音频格式不支持
- 微信播放器限制

**解决方案**:
1. 重试朗读
2. 检查网络连接
3. 使用原文模式试试

### 问题3: 翻译不准确

**症状**: 翻译结果与预期不符

**原因**:
- 免费翻译API质量有限
- 文本包含专业术语
- 上下文理解不足

**解决方案**:
1. 使用腾讯云API(质量更好)
2. 手动编辑作业内容
3. 使用原文模式
