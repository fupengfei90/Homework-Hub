# 微信同声传译插件集成指南

## 为什么需要插件?

由于微信小程序对音频播放的限制:
- ❌ 外部TTS API(有道、百度)的音频无法直接播放
- ❌ 浏览器SpeechSynthesis仅在某些环境可用
- ❌ `Unable to decode audio data` 错误

✅ **微信同声传译插件**是官方解决方案
- 支持TTS语音合成
- 专为微信小程序设计
- 稳定可靠

## 一、申请插件

### 1. 登录微信公众平台
访问: https://mp.weixin.qq.com/

### 2. 进入插件管理
1. 点击左侧菜单"设置" → "第三方设置"
2. 找到"插件管理"
3. 点击"添加插件"

### 3. 搜索插件
搜索: "同声传译"
或直接添加: `wx069ba97219f66d99`

### 4. 申请使用
1. 点击"申请使用"
2. 填写申请信息:
   - **插件使用场景**: 作业朗读
   - **使用说明**: 为学生提供作业内容朗读功能
3. 提交申请
4. 等待审核(通常1-3个工作日)

## 二、配置项目

### 1. 修改 app.json

在 `app.json` 中添加插件配置:

```json
{
  "plugins": {
    "WechatSI": {
      "version": "latest",
      "provider": "wx069ba97219f66d99"
    }
  }
}
```

### 2. 修改 page.json

在需要使用插件的页面配置中添加:

```json
{
  "usingComponents": {
    "tts-player": "plugin://WechatSI/tts-player"
  }
}
```

## 三、使用插件

### 1. 在 WXML 中使用

```xml
<view class="tts-container">
  <tts-player
    text="{{homeworkText}}"
    lang="{{lang}}"
    bind:start="onTTSStart"
    bind:end="onTTSEnd"
    bind:error="onTTSError"
  />
</view>
```

### 2. 在 JS 中使用

```javascript
Page({
  data: {
    homeworkText: '',
    lang: 'zh-CN'
  },

  onLoad(options) {
    // 获取作业内容
    const homework = this.getHomework(options.id);
    this.setData({
      homeworkText: `${homework.title}。${homework.content}`
    });
  },

  onTTSStart(e) {
    console.log('TTS开始播放', e.detail);
    this.setData({ isReading: true });
  },

  onTTSEnd(e) {
    console.log('TTS播放结束', e.detail);
    this.setData({ isReading: false });
  },

  onTTSError(e) {
    console.error('TTS播放失败', e.detail);
    wx.showToast({
      title: '播放失败',
      icon: 'none'
    });
  }
});
```

### 3. 控制播放

```javascript
// 开始播放
startTTS() {
  const ttsPlayer = this.selectComponent('#ttsPlayer');
  ttsPlayer.play();
},

// 暂停播放
pauseTTS() {
  const ttsPlayer = this.selectComponent('#ttsPlayer');
  ttsPlayer.pause();
},

// 停止播放
stopTTS() {
  const ttsPlayer = this.selectComponent('#ttsPlayer');
  ttsPlayer.stop();
}
```

## 四、集成到本项目

### 1. 修改 detail.wxml

```xml
<view class="tts-section">
  <tts-player
    id="ttsPlayer"
    text="{{readText}}"
    lang="{{ttsLang}}"
    bind:start="onTTSStart"
    bind:end="onTTSEnd"
    bind:error="onTTSError"
  />
</view>
```

### 2. 修改 detail.js

```javascript
Page({
  data: {
    readText: '',
    ttsLang: 'zh-CN'
  },

  // 开始朗读
  startReading() {
    const text = `${this.data.homework.title}。${this.data.homework.content}`;
    const readMode = this.data.readMode;
    const targetLang = this.data.targetLang;

    // 如果是翻译模式,先翻译
    if (readMode === 'translated') {
      this.translateAndSpeak(text, targetLang);
    } else {
      // 直接朗读
      this.setData({
        readText: text,
        ttsLang: 'zh-CN'
      });

      // 调用插件播放
      const ttsPlayer = this.selectComponent('#ttsPlayer');
      ttsPlayer.play();
    }
  },

  // 翻译并朗读
  async translateAndSpeak(text, targetLang) {
    // 调用翻译API
    const result = await tencentTtsService.textTranslate(text, 'zh', targetLang);

    // 设置朗读文本和语言
    this.setData({
      readText: result.TargetText,
      ttsLang: targetLang === 'en' ? 'en-US' : 'zh-CN'
    });

    // 调用插件播放
    const ttsPlayer = this.selectComponent('#ttsPlayer');
    ttsPlayer.play();
  },

  // TTS事件
  onTTSStart(e) {
    this.setData({ isReading: true });
  },

  onTTSEnd(e) {
    this.setData({ isReading: false });
  },

  onTTSError(e) {
    console.error('TTS错误:', e.detail);
    this.setData({ isReading: false });
    wx.showModal({
      title: '播放失败',
      content: '语音播放失败,请重试',
      showCancel: false
    });
  }
});
```

## 五、插件配置选项

### 支持的参数

| 参数 | 类型 | 说明 | 可选值 |
|------|------|------|--------|
| text | String | 要朗读的文本 | 任意文本 |
| lang | String | 语言代码 | zh-CN, en-US, ja-JP等 |
| speed | Number | 语速 | 0.5-2.0 |
| pitch | Number | 音调 | 0.5-2.0 |

### 事件

| 事件 | 说明 | 返回参数 |
|------|------|----------|
| bind:start | 开始播放 | - |
| bind:end | 播放结束 | - |
| bind:error | 播放错误 | error对象 |

## 六、常见问题

### 问题1: 插件未审核通过

**解决**: 提交审核后等待1-3个工作日,审核通过后即可使用

### 问题2: 插件无法加载

**检查**:
1. app.json中是否正确配置插件
2. 插件ID是否正确: `wx069ba97219f66d99`
3. 网络连接是否正常

### 问题3: TTS无法播放

**可能原因**:
- 文本为空
- 语言代码不正确
- 网络问题

**解决**:
1. 检查text参数
2. 使用正确的语言代码(zh-CN, en-US)
3. 检查网络连接
