# 图片资源说明

本目录需要包含以下图片资源文件，用于小程序的标签栏图标和空状态显示。

## 必需文件

### 标签栏图标

以下图标用于底部标签栏，建议尺寸为 **81px × 81px**（@2x），格式为 PNG。

1. **home.png** - 首页图标（未选中状态）
   - 建议使用灰色或浅色图标

2. **home-active.png** - 首页图标（选中状态）
   - 建议使用主题色（#4A90E2）图标

3. **publish.png** - 发布页图标（未选中状态）
   - 建议使用灰色或浅色图标

4. **publish-active.png** - 发布页图标（选中状态）
   - 建议使用主题色（#4A90E2）图标

### 空状态图片

5. **empty.png** - 空状态图片
   - 建议尺寸：200px × 200px
   - 用于作业列表为空时的提示

## 图标设计建议

- 使用简洁的线条图标风格
- 保持图标风格统一
- 确保图标在不同背景下清晰可见
- 可以使用图标库（如 iconfont、Font Awesome）中的图标

## 临时方案

如果暂时没有图标，可以：
1. 使用占位图片（纯色方块）
2. 使用在线图标生成工具
3. 从免费图标库下载（如 [Icons8](https://icons8.com/)）

## 注意事项

- 所有图标文件必须放在 `images/` 目录下
- 文件名必须与 `app.json` 中配置的路径一致
- 建议使用 PNG 格式，支持透明背景
